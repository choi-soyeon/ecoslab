# Personal Markdown Ready-Copy Template

Use this file as a copy source when writing `.md` content.

## 1) Basic Formatting

```md
**bold**
*italic*
***bold italic***
~~strikethrough~~
`inline code`
\*escaped literal asterisks\*
```

## 2) Headings

```md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

## 3) Links and Images

```md
[Link text](https://example.com)
[Internal page](/people/)

![Image alt text](/assets/img/your-image.jpg)
```

## 4) Lists

```md
- Bullet one
- Bullet two

1. Number one
2. Number two

- [ ] Task not done
- [x] Task done
```

## 5) Quote, Rule, and Line Break

```md
> This is a block quote.

---

First line with two spaces at the end.  
Second line
```

## 6) Code Blocks

```md
```python
# Python example
print("hello")
```
```

```md
```bash
# Bash example
ls -la
```
```

## 7) Table

```md
| Name | Role | Email |
|---|---|---|
| Person A | PI | a@example.com |
| Person B | RA | b@example.com |
```

## 8) Footnotes

```md
Here is a sentence with a footnote.[^1]

[^1]: Footnote text goes here.
```

## 9) Reusable About-Section Snippet

```md
## Research

We study how [topic] shapes [outcome] across [context].

Our methods include:
- controlled experiments
- field/observational studies
- secondary data analysis
- computational modeling
```

## 10) Reusable News Item Snippet

```md
We are recruiting graduate students and research assistants.  
See details on the [People](/people/) page.
```

## 11) Reusable Contact Snippet

```md
## Contact

Email: your.email@domain.edu  
Location: Department, University, City, Country
```
